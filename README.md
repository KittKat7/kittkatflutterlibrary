A flutter library designed to make the process adding common app features simple.

## Features

Theming: Adds a simpler way to be able to update and change the app theme.

SharedPreferences: Adds a wrapper around the SharedPreferences plugin, which simplifies getting and
setting key values.

Widgets: Adds a collection of widgets that I regularly use. These widgets usually have a custom
wrapper which has features I often use. There are also some custom widgets which have multiple
usecases.
